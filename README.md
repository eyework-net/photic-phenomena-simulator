# photic-phenomena-simulator
Web-based simulator for halo, glare, and starburst symptoms
https://eyework-net.github.io/photic-phenomena-simulator/

A browser-based simulator for visualizing and semi-quantifying **photic phenomena** such as **halo**, **glare**, and **starburst** symptoms.

Designed for ophthalmology clinics, cataract surgery counseling, premium intraocular lens (IOL) evaluation, education, and exploratory research.

## Overview

Patients often describe postoperative visual symptoms such as:

* Halo around lights
* Glare from bright sources
* Starburst patterns at night

However, these symptoms are subjective and difficult to communicate quantitatively.

This simulator was created to help patients and clinicians visually match symptom perception and generate structured records for discussion or research.

---

## Features

* Adjustable **Halo Size / Density**
* Adjustable **Glare Size / Density**
* Adjustable **Starburst Size / Density**
* Day / Night mode
* PNG image export
* CSV data export
* Browser-based (no installation required)
* Lightweight single-file HTML tool

## Disclaimer

This software is intended for **educational and research purposes only**.

It is **not a medical device**, not diagnostic software, and does not precisely reproduce any individual patient's true visual experience.

Clinical decisions should not rely solely on this simulator.
